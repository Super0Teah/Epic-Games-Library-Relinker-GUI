? How to Use
Relink Games
▾
  1. Close Epic Launcher.
  2. Ensure the Games Folder in Settings points to your library's new drive letter.
  3. Click Start Relink. The tool will automatically batch-update all manifests to match the current folder.
Move Games
▾
  1. Close Epic Launcher entirely.
  2. Select the games you want to move from the 'Move Games' list.
  3. Click 'Browse' to choose a destination, then click Move. The tool will move the files and patch the manifests for you.
Move Between PCs
▾
  1. Run this tool on your old PC. It will automatically back up all manifests.
  2. When prompted, unplug your game drive and connect it to the new PC.
  3. Run the app on the new PC and click OK to instantly restore all game paths.
Capture
▾
  1. Click Start Capture to automatically scan for games missing a manifest.
  2. For each game, the tool will prompt you to start a download in Epic and immediately cancel it.
  3. The tool will automatically intercept the newly generated `.item` file and save it.
Link
▾
  1. Close Epic Launcher entirely.
  2. Click Start Link to find pending/intercepted manifests.
  3. Select the correct game folder for each manifest to permanently register it.
Fix Games (manifests)
▾
  1. Close Epic Launcher entirely.
  2. Click Start Fix to load all active manifests.
  3. Select a broken game manifest, select its correct folder, and click Fix Link.
Fix DLCs (Manifests)
▾
  1. Close Epic Launcher entirely.
  2. Click Start Fix DLCs to view all manifests.
  3. Select the DLC's broken manifest, select its Base Game folder, and click Fix Link.
Library Hub
▾
  1. Click Refresh Library to scan your installed games.
  2. If a game shows a "Path Broken" status, use the Fix tools above.
  3. Click the Play button to launch a game directly from the app.
Manifest Tools
▾

🧹 Manifest Cleanup

  1. Click Manifest Cleanup to scan for ghost/orphaned .item files.
  2. Select the entries you want to remove and click Delete Selected.
  3. Restart the Epic Games Launcher to clear ghost games from the library.

✔ Manifest Validator

  1. Click Manifest Validator to check all manifests for structural issues.
  2. Warnings indicate broken paths; errors indicate corrupt/unreadable files.
  3. Use Fix Manifest Link to repair any broken path references.
