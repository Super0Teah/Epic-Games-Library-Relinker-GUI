? How to Use
Relink & Move
▾

Relink All

  1. Point Games Folder to your new library location.
  2. Click Relink All to batch-patch all manifests at once.

Move Games

  1. Select games, choose a destination, and click Move. The tool handles the file transfer and manifest patching.
Library Hub
▾
  1. Click Refresh Library to scan your installed games.
  2. If a game says Unregistered, click Import to sync it with Epic.
  3. Restart the Epic Launcher when prompted to see the changes.
Manifest Maintenance
▾

🧹 Manifest Cleanup

  1. Removes "ghost" game entries and duplicate download files.

✔ Manifest Validator

  1. Checks all manifests for structural errors and broken path references.
Advanced Tools
▾

Capture & Link

  1. Use these only if Import fails or if no .mancpn metadata exists.

Fix Manifest Link

  1. Allows manual folder-to-manifest matching using the interactive picker.
